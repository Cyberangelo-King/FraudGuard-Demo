Model Architecture

Stacking Ensemble
Two-layer heterogeneous stacking. Base learners generate out-of-fold predictions that form the meta-features for the final classifier.
📥
Input
28 PCA-transformed features + Time + Amount
↓
🧩
Layer 1 — Base Learners
Logistic Regression · Random Forest · XGBoost
↓
🔗
Layer 2 — Meta-Learner
Logistic Regression on base learner probability outputs
↓
🎯
Output
Fraud probability score (0–1) → Threshold 0.682 → Fraud / Legitimate
